## 🎯 How to Get Started

### Immediate Use (Right Now)
```bash
npm start
```
Then click the orange chat button that appears!

### Customize Colors (Optional - 2 minutes)
Find `rgb(240, 82, 4)` in `Chatbot.css` and replace with your color.

### Customize Size (Optional - 1 minute)
Edit the `width` property in `.chatbot-container` in `Chatbot.css`.

### Change Response Length (Optional - 1 minute)
Edit the `maxLength: 300` parameter in the `truncateResponse()` function in `Chatbot.js`.
