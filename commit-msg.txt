feat/fix: Refactor AI page into chatbot UI, fix markdown rendering, and redesign learning path sidebars

This commit introduces significant UI improvements across the Learning Paths and the AI assistant page, alongside a critical fix for markdown rendering bugs.

### 1. AI Page Markdown Rendering Fix (RCA)
- **Root Cause Analysis**: The AI response previously had broken layouts (massive code blocks, tables rendering as raw text). The root cause was the upgrade to `react-markdown` v10, which silently removed the `inline` prop from the `code` component, causing all inline backtick strings to render as full-width block elements. Additionally, the AI (Groq) was emitting `<br>` HTML tags inside tables which the default parser treats as literal text.
- **Fix**: 
  - Restructured the `code` and `pre` component overrides in `Ai.js` to properly identify block vs. inline code using the v10 pattern (checking for `className` or newline characters).
  - Installed `rehype-raw` plugin to parse standard HTML emitted by the AI.
  - Implemented a `preprocessMarkdown()` hook to normalize `<br>` tags and collapse excessive empty lines.
  - Completely restyled tables using Tailwind utility classes, adding zebra-striping (`even:bg-slate-50/80`) for drastically improved readability on large data tables.

### 2. AI Page Chatbot UI Refactoring
- Transformed `Ai.js` from a static form page into a dynamic, conversational chat layout.
- **Dynamic View State**: Initially shows a large center prompt. Upon submission, transitions smoothly into a chat interface.
- **Chat Bubbles**: Renders the user's query as a blue, right-aligned chat bubble and the AI's markdown response as a clean, left-aligned bubble.
- **Sticky Input Form**: Introduced a sticky input bar at the bottom of the screen to allow rapid follow-up queries without forcing the user to scroll back to the top.
- Moved the "Clear Chat" and "Ask" buttons to the bottom sticky bar for better ergonomics.

### 3. Learning Paths Sidebar Redesign
- Applied a unified, dark-themed redesign across all 6 learning path themes (`Fundamentals.css`, `JsCore.css`, `Frontend.css`, `NodeJs.css`, `FullStack.css`, `TechTrends.css`).
- Broadened the sidebar width from 300px to 340px.
- Replaced the light background with a premium dark charcoal gradient (`linear-gradient(#1a1f2e, #0f1520)`) to cleanly separate the navigation from the white lesson content area.
- Updated text colors and progress indicators to ensure high contrast against the dark background.
- Cleaned up the sidebar items in `LearningPathTopic.js` by removing the preview description text (`sublink-preview`), keeping only the topic headers for faster scanning.
