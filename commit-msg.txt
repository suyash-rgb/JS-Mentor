fix: Chevron Tracker auto-advance 

when you successfully submit a solution, the system now instantly auto-advances the tracker (and the UI) to the next challenge if one exists.

The chevron color wasn't turning green right away: Because we were still "active" on that exercise tab, the CSS for the active (blue) state was actively overriding the completed (green) state.

Now, if you submit a solution:

It turns green instantly.
It immediately slides one over to the next un-solved exercise (revealing its card).
If one were to click back to a solved exercise, it will stay green so they clearly know it's already complete!
