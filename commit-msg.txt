integrate: path-based filtering for quizzes

Smart Filtering:
 - Dropdown UI: Added a sleek "Filter by Path" dropdown on the top right of the Quiz card.
 - Dynamic Options: The filter is automatically populated with the learning paths from your curriculum (e.g., Fundamentals, JS Core).
 - API Integration: Choosing a path triggers a specialized request to /curriculum/quizzes?path_heading=..., instantly narrowing down the list.

Layout Optimization:
 - Full-Width Cards: Updated the quiz list items to explicitly take 100% width. This prevents the "fragmented" look where cards were varying in size based on their text content.
Responsive Scaling: The cards now stretch elegantly across the entire content area, providing a more professional and symmetric dashboard feel.

Architecture Refactoring:
Refactored CurriculumManager.js to fetch Learning Path names at the parent level. This ensures that both the Syllabus and Quiz tabs always use the same synchronized list of topics without redundant API calls.
