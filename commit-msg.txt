feat: add images in notes

Installed library: Added browser-image-compression to handle client-side image compression.
Updated Toolbar: Added an Image icon next to the other formatting buttons in the Notes Editor.

Compression Logic:
Files are dynamically compressed using a web worker to prevent UI freezing.
The strict limit you requested (maxSizeMB: 0.25) has been applied, so images will be downscaled and compressed to sit at or below 250KB.
The maximum resolution is capped at 1280px on the longest edge, further reducing unnecessary payload bloat.

Base64 Injection:
The compressed image is seamlessly converted to a Base64 string.
The standard markdown syntax ![Image](data:image/jpeg;base64,...) is injected straight into the text box.
