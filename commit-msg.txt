feat: Syllabus / Learning Path Configuration

1. New Standalone "Syllabus Editor" (New Tab & Dedicated Route)
Stripped the limited form out of the CurriculumManager.js file and replaced it with a prompt to Launch Syllabus Editor. Clicking this will open an entirely new browser tab (at /trainer/curriculum/editor?path=Fundamentals)!

2. Complex Structure Mirroring data.json
The new SyllabusEditor component gives trainers the full power to manipulate the exact schema elements stored in data.json:

- Path Config: Path Title & Overview.
- Topics Management: Add/remove multiple topics, manage URLs (e.g. /intro), and write the introductory descriptions.
- Deep Content Sections: Inside each Topic, trainers can now dynamically click "Add Section", allowing them to manage nested elements:
  - titleX: Section Titles
  - paraX: Paragraph body text
  - codeX: Embedded code examples (formats as a monospace terminal-block)
  - headingXSubheadingY: Bullet point lists inside sections.

3. Real-Time Live Preview
Instead of guessing what the page will look like, the editor has a split-screen design:

- Left Pane: Dynamic Configuration form.
- Right Pane: A Live Preview Engine. As the trainer types a title, adds a description, injects a code example, or configures bullet points, it immediately visually renders on the right side using the exact same mapping logic we use for the students (FundamentalsTopic.js / JsCoreTopic.js UI rendering style).
